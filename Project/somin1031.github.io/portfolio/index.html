交通安全BOOK

信号のない横断歩道で、歩行者を意識せず通っていく車が多いと思っていたので、表紙で横断歩道を大きく使い、子供向けのような既存の表紙から大きく変わるよう意識した。

チョコレートパッケージ

１日の終わりに香りを嗅いで落ち着きたい。そんな時にあったら嬉しいチョコレートをイメージした。デパ地下での販売を想定し、４種類の色と女性をイメージした線画を用いてデザインした。

星の王子さま

王子様はいろんな星の人たちとの交流を経て、心が成長していく描写が物語から見えたので、王子様と星の人たちを表裏で重ね合わせるブックデザインにした。

パンフレットデザイン

デザインする前に、就労支援をおこなっている事業所のパンフレットを収集した結果似ているものがほとんどだった。そこで、クライアントの特色が表現できるようデザインした。

うちわ

蝿の王ブックカバー

物語の中で、権力の象徴になっていた法螺貝は物語が進むと法螺貝を壊すことで、人が野生に帰っていくという流れから、表紙の法螺貝が帯を開くと崩れていくデザインで表現した。
